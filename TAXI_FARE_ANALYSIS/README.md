🚖 Taxi Fare Analysis – Power BI Dashboard

📌 Project Overview
This project analyzes taxi fare data to uncover key insights into pricing patterns, trip behavior, and revenue trends. The interactive Power BI dashboard provides a visual exploration of the dataset, enabling stakeholders to make data‑driven decisions about fare optimization, customer demand, and operational efficiency.

🎯 Objectives
* Understand fare distribution across different trip distances and durations
* Identify peak hours, busiest days, and seasonal demand patterns
* Compare fare trends by location and passenger count
* Provide actionable insights for pricing strategies and service improvements
  
  🛠️ Tools & Technologies
* Power BI – Dashboard design, data visualization, DAX measures
* Excel / CSV – Raw dataset storage and preprocessing
* Data Cleaning – Handling missing values, correcting anomalies, formatting columns
* DAX Functions – Custom calculations for KPIs and measures
  
  📊 Dashboard Features
* Fare Distribution Analysis – Histogram and box plots for fare ranges
* Trip Insights – Average fare by distance, duration, and passenger count
* Time Trends – Hourly, daily, and monthly demand visualization
* Revenue KPIs – Total revenue, average fare per trip, and top‑earning routes

Interactive Filters – Drill‑down by date, location, and trip characteristics
